зЁӢеәҸи®Ўж•°еҷЁ

зЁӢеәҸи®Ўж•°еҷЁжҳҜдёҖеқ—иҫғе°Ҹзҡ„еҶ…еӯҳз©әй—ҙпјҢе®ғеҸҜд»ҘзңӢдҪңжҳҜеҪ“еүҚзәҝзЁӢжүҖжү§иЎҢзҡ„еӯ—иҠӮз Ғзҡ„иЎҢеҸ·жҢҮзӨәеҷЁгҖӮеңЁ Java иҷҡжӢҹжңәзҡ„жҰӮеҝөжЁЎеһӢйҮҢпјҢеӯ—иҠӮз Ғи§ЈйҮҠеҷЁе·ҘдҪңж—¶е°ұжҳҜйҖҡиҝҮж”№еҸҳиҝҷдёӘи®Ўж•°еҷЁзҡ„еҖјжқҘйҖүеҸ–дёӢдёҖжқЎйңҖиҰҒжү§иЎҢзҡ„еӯ—иҠӮз ҒжҢҮд»ӨпјҢе®ғжҳҜзЁӢеәҸжҺ§еҲ¶жөҒзҡ„жҢҮзӨәеҷЁпјҢеҲҶж”ҜгҖҒеҫӘзҺҜгҖҒи·іиҪ¬гҖҒејӮеёёеӨ„зҗҶгҖҒзәҝзЁӢжҒўеӨҚзӯүеҹәзЎҖеҠҹиғҪйғҪйңҖиҰҒдҫқиө–иҝҷдёӘи®Ўж•°еҷЁжқҘе®ҢжҲҗгҖӮ
з”ұдәҺ Java иҷҡжӢҹжңәзҡ„еӨҡзәҝзЁӢжҳҜйҖҡиҝҮзәҝзЁӢиҪ®жөҒеҲҮжҚўгҖҒеҲҶй…ҚеӨ„зҗҶеҷЁжү§иЎҢж—¶й—ҙзҡ„ж–№ејҸжқҘе®һзҺ°зҡ„пјҢеңЁд»»дҪ•дёҖдёӘзЎ®е®ҡзҡ„ж—¶еҲ»пјҢдёҖдёӘеӨ„зҗҶеҷЁпјҲеҜ№дәҺеӨҡж ёеӨ„зҗҶеҷЁжқҘиҜҙжҳҜдёҖдёӘеҶ…ж ёпјүйғҪеҸӘдјҡжү§иЎҢдёҖжқЎзәҝзЁӢдёӯзҡ„жҢҮд»ӨгҖӮеӣ жӯӨпјҢдёәдәҶзәҝзЁӢеҲҮжҚўеҗҺиғҪжҒўеӨҚеҲ°жӯЈзЎ®зҡ„жү§иЎҢдҪҚзҪ®пјҢжҜҸжқЎзәҝзЁӢйғҪйңҖиҰҒжңүдёҖдёӘзӢ¬з«Ӣзҡ„зЁӢеәҸи®Ўж•°еҷЁпјҢеҗ„жқЎзәҝзЁӢд№Ӣй—ҙи®Ўж•°еҷЁдә’дёҚеҪұе“ҚпјҢзӢ¬з«ӢеӯҳеӮЁпјҢжҲ‘д»¬з§°иҝҷзұ»еҶ…еӯҳеҢәеҹҹдёәвҖңзәҝзЁӢз§ҒжңүвҖқзҡ„еҶ…еӯҳгҖӮ
еҰӮжһңзәҝзЁӢжӯЈеңЁжү§иЎҢзҡ„жҳҜдёҖдёӘ Java ж–№жі•пјҢиҝҷдёӘи®Ўж•°еҷЁи®°еҪ•зҡ„жҳҜжӯЈеңЁжү§иЎҢзҡ„иҷҡжӢҹжңәеӯ—иҠӮз ҒжҢҮд»Өзҡ„ең°еқҖпјӣеҰӮжһңжӯЈеңЁжү§иЎҢзҡ„жҳҜжң¬ең°пјҲNativeпјүж–№жі•пјҢиҝҷдёӘи®Ўж•°еҷЁеҖјеҲҷеә”дёәз©әпјҲUndefinedпјүгҖӮжӯӨеҶ…еӯҳеҢәеҹҹжҳҜе”ҜдёҖдёҖдёӘеңЁгҖҠJava иҷҡжӢҹжңәи§„иҢғгҖӢдёӯжІЎжңүи§„е®ҡд»»дҪ•OutOfMemoryError жғ…еҶөзҡ„еҢәеҹҹгҖӮ

Java иҷҡжӢҹжңәж Ҳ

дёҺзЁӢеәҸи®Ўж•°еҷЁдёҖж ·пјҢJava иҷҡжӢҹжңәж Ҳд№ҹжҳҜзәҝзЁӢз§Ғжңүзҡ„пјҢе®ғзҡ„з”ҹе‘Ҫе‘ЁжңҹдёҺзәҝзЁӢзӣёеҗҢгҖӮиҷҡжӢҹжңәж ҲжҸҸиҝ°зҡ„жҳҜ Java ж–№жі•жү§иЎҢзҡ„зәҝзЁӢеҶ…еӯҳжЁЎеһӢпјҡжҜҸдёӘж–№жі•иў«жү§иЎҢзҡ„ж—¶еҖҷпјҢJava иҷҡжӢҹжңәйғҪдјҡеҗҢжӯҘеҲӣе»әдёҖдёӘж Ҳеё§пјҲStack Frameпјүз”ЁдәҺеӯҳеӮЁеұҖйғЁеҸҳйҮҸиЎЁгҖҒж“ҚдҪңж•°ж ҲгҖҒеҠЁжҖҒиҝһжҺҘгҖҒж–№жі•еҮәеҸЈзӯүдҝЎжҒҜгҖӮжҜҸдёҖдёӘж–№жі•иў«и°ғз”ЁзӣҙиҮіжү§иЎҢе®ҢжҜ•зҡ„иҝҮзЁӢпјҢе°ұеҜ№еә”зқҖдёҖдёӘж Ҳеё§еңЁиҷҡжӢҹжңәж Ҳдёӯд»Һе…Ҙж ҲеҲ°еҮәж Ҳзҡ„иҝҮзЁӢгҖӮ
еұҖйғЁеҸҳйҮҸиЎЁеӯҳж”ҫдәҶзј–иҜ‘жңҹеҸҜзҹҘзҡ„еҗ„з§Қ Java иҷҡжӢҹжңәеҹәжң¬ж•°жҚ®зұ»еһӢпјҲbooleanгҖҒbyteгҖҒcharгҖҒshortгҖҒintгҖҒ floatгҖҒlongгҖҒdoubleпјүгҖҒеҜ№иұЎеј•з”Ёе’Ң returnAddress зұ»еһӢпјҲжҢҮеҗ‘дәҶдёҖжқЎеӯ—иҠӮз ҒжҢҮд»Өзҡ„ең°еқҖпјүгҖӮ
иҝҷдәӣж•°жҚ®зұ»еһӢеңЁеұҖйғЁеҸҳйҮҸиЎЁдёӯзҡ„еӯҳеӮЁз©әй—ҙд»ҘеұҖйғЁеҸҳйҮҸж§ҪпјҲSlotпјүжқҘиЎЁзӨәпјҢе…¶дёӯ 64 дҪҚй•ҝеәҰзҡ„ long е’Ң double зұ»еһӢзҡ„ж•°жҚ®дјҡеҚ з”ЁдёӨдёӘеҸҳйҮҸж§ҪпјҢе…¶дҪҷзҡ„ж•°жҚ®зұ»еһӢеҸӘеҚ з”ЁдёҖдёӘгҖӮеұҖйғЁеҸҳйҮҸиЎЁжүҖйңҖзҡ„еҶ…еӯҳз©әй—ҙеңЁзј–иҜ‘жңҹй—ҙе®ҢжҲҗеҲҶй…ҚпјҢеҪ“иҝӣе…ҘдёҖдёӘж–№жі•ж—¶пјҢиҝҷдёӘж–№жі•йңҖиҰҒеңЁж Ҳеё§дёӯеҲҶй…ҚеӨҡеӨ§зҡ„еұҖйғЁеҸҳйҮҸз©әй—ҙжҳҜе®Ңе…ЁзЎ®е®ҡзҡ„пјҢеңЁж–№жі•иҝҗиЎҢжңҹй—ҙдёҚдјҡж”№еҸҳеұҖйғЁеҸҳйҮҸиЎЁзҡ„еӨ§е°ҸгҖӮ
иҷҡжӢҹжңәж ҲдёӨзұ»ејӮеёёзҠ¶еҶөпјҡеҰӮжһңзәҝзЁӢиҜ·жұӮзҡ„ж Ҳж·ұеәҰеӨ§дәҺиҷҡжӢҹжңәжүҖе…Ғи®ёзҡ„ж·ұеәҰпјҢе°ҶжҠӣеҮә StackOverflowError ејӮеёёпјӣеҰӮжһң Java иҷҡжӢҹжңәж Ҳе®№йҮҸеҸҜд»ҘеҠЁжҖҒжү©еұ•пјҢеҪ“ж Ҳжү©еұ•ж—¶ж— жі•з”іиҜ·еҲ°и¶іеӨҹзҡ„еҶ…еӯҳдјҡжҠӣеҮә OutOfMemoryError ејӮеёёгҖӮ

жң¬ең°ж–№жі•ж Ҳ

жң¬ең°ж–№жі•ж ҲдёҺиҷҡжӢҹжңәж ҲжүҖеҸ‘жҢҘзҡ„дҪңз”ЁжҳҜйқһеёёзӣёдјјзҡ„пјҢе…¶еҢәеҲ«еҸӘжҳҜиҷҡжӢҹжңәж ҲдёәиҷҡжӢҹжңәжү§иЎҢ Java ж–№жі•пјҲд№ҹе°ұжҳҜеӯ—иҠӮз ҒпјүжңҚеҠЎпјҢиҖҢжң¬ең°ж–№жі•ж ҲеҲҷжҳҜдёәиҷҡжӢҹжңәдҪҝз”ЁеҲ°зҡ„жң¬ең°пјҲNativeпјүж–№жі•жңҚеҠЎгҖӮ
жң¬ең°ж–№жі•ж Ҳд№ҹдјҡеңЁж Ҳж·ұеәҰжәўеҮәжҲ–иҖ…ж Ҳжү©еұ•еӨұиҙҘж—¶еҲҶеҲ«жҠӣеҮә StackOverflowError е’ҢOutOfMemoryError ејӮеёёгҖӮ

Java е Ҷ

Java е ҶжҳҜиҷҡжӢҹжңәжүҖз®ЎзҗҶзҡ„еҶ…еӯҳдёӯжңҖеӨ§зҡ„дёҖеқ—гҖӮJava е ҶжҳҜиў«жүҖжңүзәҝзЁӢе…ұдә«зҡ„дёҖеқ—еҶ…еӯҳеҢәеҹҹпјҢеңЁиҷҡжӢҹжңәеҗҜеҠЁж—¶еҲӣе»әгҖӮжӯӨеҶ…еӯҳеҢәеҹҹзҡ„е”ҜдёҖзӣ®зҡ„е°ұжҳҜеӯҳж”ҫеҜ№иұЎе®һдҫӢпјҢJava йҮҢвҖңеҮ д№ҺвҖқжүҖжңүзҡ„еҜ№иұЎе®һдҫӢйғҪеңЁиҝҷйҮҢеҲҶй…ҚеҶ…еӯҳгҖӮ
Java е ҶжҳҜеһғеңҫж”¶йӣҶеҷЁз®ЎзҗҶзҡ„еҶ…еӯҳеҢәеҹҹгҖӮ

ж–№жі•еҢә

ж–№жі•еҢәдёҺ Java е ҶдёҖж ·пјҢжҳҜеҗ„дёӘзәҝзЁӢе…ұдә«зҡ„еҶ…еӯҳеҢәеҹҹпјҢе®ғз”ЁдәҺеӯҳеӮЁе·Іиў«иҷҡжӢҹжңәеҠ иҪҪзҡ„зұ»еһӢдҝЎжҒҜгҖҒеёёйҮҸгҖҒйқҷжҖҒеҸҳйҮҸгҖҒеҚіж—¶зј–иҜ‘еҷЁзј–иҜ‘еҗҺзҡ„д»Јз Ғзј“еӯҳзӯүж•°жҚ®гҖӮиҷҪз„¶ж–№жі•еҢәиў«жҸҸиҝ°дёәе Ҷзҡ„дёҖдёӘйҖ»иҫ‘йғЁеҲҶпјҢдҪҶжҳҜе®ғеҚҙжңүдёҖдёӘеҲ«еҗҚеҸ«дҪңвҖңйқһе ҶвҖқпјҲNon-HeapпјүпјҢзӣ®зҡ„жҳҜдёҺ Java е ҶеҢәеҲҶејҖжқҘгҖӮ
еңЁ JDK 8 д»ҘеүҚпјҢHotSpot иҷҡжӢҹжңәжҠҠж”¶йӣҶеҷЁзҡ„еҲҶд»Ји®ҫи®Ўжү©еұ•иҮіж–№жі•еҢәпјҢжҲ–иҖ…иҜҙдҪҝз”Ёж°ёд№…д»ЈжқҘе®һзҺ°ж–№жі•еҢәпјҢиҝҷж ·дҪҝеҫ— HotSpot зҡ„еһғеңҫж”¶йӣҶеҷЁиғҪеӨҹеғҸз®ЎзҗҶ Java е ҶдёҖж ·з®ЎзҗҶиҝҷйғЁеҲҶеҶ…еӯҳпјҢзңҒеҺ»дё“й—Ёдёәж–№жі•еҢәзј–еҶҷеҶ…еӯҳз®ЎзҗҶд»Јз Ғзҡ„е·ҘдҪңпјҢдҪҶиҝҷз§Қи®ҫи®ЎеҜјиҮҙдәҶ Java еә”з”Ёжӣҙе®№жҳ“йҒҮеҲ°еҶ…еӯҳжәўеҮәзҡ„й—®йўҳгҖӮ
еңЁ JDK 6 зҡ„ж—¶еҖҷ HotSpot е°ұжңүж”ҫејғж°ёд№…д»ЈпјҢйҖҗжӯҘж”№дёәйҮҮз”Ёжң¬ең°еҶ…еӯҳжқҘе®һзҺ°ж–№жі•еҢәзҡ„и®ЎеҲ’дәҶпјҢеҲ°дәҶ JDK 7 зҡ„ HotSpotпјҢе·Із»ҸжҠҠеҺҹжң¬ж”ҫеңЁж°ёд№…д»Јзҡ„еӯ—з¬ҰдёІеёёйҮҸжұ гҖҒйқҷжҖҒеҸҳйҮҸзӯүз§»еҮәпјҢиҖҢеҲ°дәҶ JDK 8пјҢз»ҲдәҺе®Ңе…ЁеәҹејғдәҶж°ёд№…д»Јзҡ„жҰӮеҝөпјҢж”№з”ЁеңЁжң¬ең°еҶ…еӯҳдёӯе®һзҺ°зҡ„е…ғз©әй—ҙжқҘд»ЈжӣҝпјҢжҠҠ JDK 7 дёӯж°ёд№…д»Јиҝҳеү©дҪҷзҡ„еҶ…е®№пјҲдё»иҰҒжҳҜзұ»еһӢдҝЎжҒҜпјүе…ЁйғЁз§»еҲ°е…ғз©әй—ҙдёӯгҖӮ
еҜ№ж–№жі•еҢәзҡ„зәҰжқҹйқһеёёе®ҪжқҫпјҢйҷӨдәҶе’Ң Java е ҶдёҖж ·дёҚйңҖиҰҒиҝһз»ӯзҡ„еҶ…еӯҳе’ҢеҸҜд»ҘйҖүжӢ©еӣәе®ҡеӨ§е°ҸжҲ–иҖ…еҸҜжү©еұ•еӨ–пјҢз”ҡиҮіиҝҳеҸҜд»ҘйҖүжӢ©дёҚе®һзҺ°еһғеңҫж”¶йӣҶгҖӮеһғеңҫж”¶йӣҶиЎҢдёәеңЁиҝҷдёӘеҢәеҹҹжҳҜжҜ”иҫғе°‘еҮәзҺ°зҡ„пјҢиҝҷеҢәеҹҹзҡ„еҶ…еӯҳеӣһж”¶зӣ®ж Үдё»иҰҒжҳҜй’ҲеҜ№еёёйҮҸжұ зҡ„еӣһж”¶е’ҢеҜ№зұ»еһӢзҡ„еҚёиҪҪгҖӮ
еҰӮжһңж–№жі•еҢәж— жі•ж»Ўи¶іж–°зҡ„еҶ…еӯҳеҲҶй…ҚйңҖжұӮж—¶пјҢе°ҶжҠӣеҮә OutOfMemoryError ејӮеёёгҖӮ

иҝҗиЎҢж—¶еёёйҮҸжұ 

иҝҗиЎҢж—¶еёёйҮҸжұ жҳҜж–№жі•еҢәзҡ„дёҖйғЁеҲҶгҖӮClass ж–Үд»¶дёӯйҷӨдәҶжңүзұ»зҡ„зүҲжң¬гҖҒеӯ—ж®өгҖҒж–№жі•гҖҒжҺҘеҸЈзӯүжҸҸиҝ°дҝЎжҒҜеӨ–пјҢиҝҳжңүдёҖйЎ№дҝЎжҒҜжҳҜеёёйҮҸжұ иЎЁпјҢз”ЁдәҺеӯҳж”ҫзј–иҜ‘жңҹз”ҹжҲҗзҡ„еҗ„з§Қеӯ—йқўйҮҸдёҺз¬ҰеҸ·еј•з”ЁпјҢиҝҷйғЁеҲҶеҶ…е®№е°ҶеңЁзұ»еҠ иҪҪеҗҺеӯҳж”ҫеҲ°ж–№жі•еҢәзҡ„иҝҗиЎҢж—¶еёёйҮҸжұ дёӯгҖӮ
иҝҗиЎҢж—¶еёёйҮҸжұ зӣёеҜ№дәҺ Class ж–Үд»¶еёёйҮҸжұ зҡ„еҸҰеӨ–дёҖдёӘйҮҚиҰҒзү№еҫҒжҳҜе…·еӨҮеҠЁжҖҒжҖ§пјҢJava иҜӯиЁҖе№¶дёҚиҰҒжұӮеёёйҮҸдёҖе®ҡеҸӘжңүзј–иҜ‘жңҹжүҚиғҪдә§з”ҹпјҢд№ҹе°ұжҳҜиҜҙпјҢе№¶йқһйў„зҪ®е…Ҙ Class ж–Үд»¶дёӯеёёйҮҸжұ зҡ„еҶ…е®№жүҚиғҪиҝӣе…Ҙж–№жі•еҢәиҝҗиЎҢж—¶еёёйҮҸжұ пјҢиҝҗиЎҢжңҹй—ҙд№ҹеҸҜд»Ҙе°Ҷж–°зҡ„еёёйҮҸж”ҫе…Ҙжұ дёӯпјҢиҝҷз§Қзү№жҖ§иў«ејҖеҸ‘дәәе‘ҳеҲ©з”Ёеҫ—жҜ”иҫғеӨҡзҡ„дҫҝжҳҜ String зұ»зҡ„ intern()ж–№жі•гҖӮ
ж—ўз„¶иҝҗиЎҢж—¶еёёйҮҸжұ жҳҜж–№жі•еҢәзҡ„дёҖйғЁеҲҶпјҢиҮӘз„¶еҸ—еҲ°ж–№жі•еҢәеҶ…еӯҳзҡ„йҷҗеҲ¶пјҢеҪ“еёёйҮҸжұ ж— жі•еҶҚз”іиҜ·еҲ°еҶ…еӯҳж—¶дјҡжҠӣеҮә OutOfMemoryError ејӮеёёгҖӮ

зӣҙжҺҘеҶ…еӯҳ

зӣҙжҺҘеҶ…еӯҳе№¶дёҚжҳҜиҷҡжӢҹжңәиҝҗиЎҢж—¶ж•°жҚ®еҢәеҹҹзҡ„дёҖйғЁеҲҶпјҢд№ҹдёҚжҳҜгҖҠи§„иҢғгҖӢдёӯе®ҡд№үзҡ„еҶ…еӯҳеҢәеҹҹгҖӮдҪҶжҳҜиҝҷйғЁеҲҶеҶ…еӯҳд№ҹиў«йў‘з№Ғең°дҪҝз”ЁпјҢиҖҢдё”д№ҹеҸҜиғҪеҜјиҮҙOutOfMemoryError ејӮеёёеҮәзҺ°гҖӮ
еңЁ JDK 1.4 дёӯж–°еҠ е…ҘдәҶ NIOзұ»пјҢеј•е…ҘдәҶдёҖз§ҚеҹәдәҺйҖҡйҒ“пјҲChannelпјүдёҺзј“еҶІеҢәпјҲBufferпјүзҡ„ I/O ж–№ејҸпјҢе®ғеҸҜд»ҘдҪҝз”Ё Native еҮҪж•°еә“зӣҙжҺҘеҲҶй…Қе ҶеӨ–еҶ…еӯҳпјҢз„¶еҗҺйҖҡиҝҮдёҖдёӘеӯҳеӮЁеңЁ Java е ҶйҮҢйқўзҡ„ DirectByteBuffer еҜ№иұЎдҪңдёәиҝҷеқ—еҶ…еӯҳзҡ„еј•з”ЁиҝӣиЎҢж“ҚдҪңгҖӮиҝҷж ·иғҪеңЁдёҖдәӣеңәжҷҜдёӯжҳҫи‘—жҸҗй«ҳжҖ§иғҪпјҢеӣ дёәйҒҝе…ҚдәҶеңЁ Java е Ҷе’Ң Native е ҶдёӯжқҘеӣһеӨҚеҲ¶ж•°жҚ®гҖӮ
й…ҚзҪ®иҷҡжӢҹжңәеҸӮж•°ж—¶пјҢдјҡж №жҚ®е®һйҷ…еҶ…еӯҳеҺ»и®ҫзҪ®-XmxзӯүеҸӮж•°дҝЎжҒҜпјҢдҪҶз»ҸеёёеҝҪз•ҘжҺүзӣҙжҺҘеҶ…еӯҳпјҢдҪҝеҫ—еҗ„дёӘеҶ…еӯҳеҢәеҹҹжҖ»е’ҢеӨ§дәҺзү©зҗҶеҶ…еӯҳйҷҗеҲ¶пјҢд»ҺиҖҢеҜјиҮҙеҠЁжҖҒжү©еұ•ж—¶еҮәзҺ° OutOfMemoryError ејӮеёёгҖӮ
